# HR Analytics Dashboard – Employee Attrition Analyis

## Project Description
This project analyzes employee data to identify attrition patterns, workforce distribution, and job satisfaction levels. The dashboard helps understand which departments, age groups, and education fields experience higher employee turnover, enabling better HR decision-making.

## Dashboard Preview
![HR Analytics Dashboard](HR%20Dashboard.png)

## Key Insights
- The organization has 1,470 total employees, with 237 employees leaving, resulting in a 16% attrition rate.
- Sales department shows th highest attrition compared to R&D and HR.
- Most employees belong to the 30-40 age group.
- The 25-34 age group experience the highest attrition.
- Male employees have a higher attrition count then female employees.
- Life Sciences and Medical education fields show higher employee turnover.

## Tools Used
- Tableau -> Dashboard creation and data visualization.
- Microsoft Excel -> CSV Dataset - Data source.
- Data Cleaning & Preparation -> Data preprocessing before visualization.
