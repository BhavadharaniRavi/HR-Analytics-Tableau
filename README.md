# HR Analytics Dashboard – Tableau

## Project Overview
This project presents an HR Analytics dashboard built using Tableau to analyze employee data and key HR metrics.

## Key Insights
- Employee attrition analysis
- Department-wise employee distribution
- Workforce demographics
- Key HR performance indicators

## Tools Used
- Tableau
- Data Visualization
- HR Analytics

## Project File
The Tableau packaged workbook (.twbx) file is included in this repository.
